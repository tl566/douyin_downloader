# v4.9.8

目前官方调整了福袋的获取频率。
过去每个账号两个小时内可以抢满8-12个福袋，经过测试无论是扭蛋机还是其他福袋软件都无法很快抢满福袋了。
当然，号与号之间也有区别，优质号受影响较小，普通号受影响较大。
经过几天的测试，我们的建议是：
	1.增加抢福袋的挂机时长，延长至8-16小时中奖的次数还是很不错的。这是官方的改动，软件确实也没有办法规避这个问题。
	2.在被风控的时间内，建议每天挂机养号，增加账号活跃权重，避免被识别为工作室账号，否则中奖率必然会受到影响。
	3.新版本增加了抢福袋的时间间隔，之前的抢福袋频率差不多是每1~2分钟1个福袋的频率，这个频率过高可能会被官方标记为工作室行为，大家可以设置一下间隔，让频繁抢福袋的行为不至于那么明显。

新增加了“不显示直播”功能，使抢福袋时不耗费额外的流量，减少系统资源的开销。


# v4.9.7

解决了特定情况下，粉丝团福袋类型判断错误，导致不加入粉丝团的bug。

增加“只抢粉丝团福袋”功能，建议vip用户使用该功能。
每个粉丝团福袋虽然会产生1钻的付费行为，但在一定程度上可以防止1-3天的风控时间，这样每天都可以抢到福袋，让收益增加不少。
每个人的账号活跃度不一样，所以此功能每个账号收益增加的情况也不一样，需要大家根据自己的情况测试，然后每日定量去抢，这样既可以增加收益，也可以节省加入粉丝团的费用。
请在使用该功能之前，务必在自己的账号中预留一点钻石，用来赠送自动加入粉丝团所需要的礼物。

增加“停止条件”功能，可以控制抢福袋的总时长和总福袋个数。满足条件任意一个，扭蛋机将停止工作。设置好抢福袋的个数和抢福袋的时长，点击“停止条件”按钮即可设置成功。
强烈建议vip用户将此功能与“只抢粉丝团福袋”功能配合使用，这样可以定时定量的去抢粉丝团福袋，避免产生不必要的加入粉丝团的费用。

修复“固定”直播间扫描不到福袋的bug。vip用户可以最多监听扫描10个固定直播间福袋。免费版限制监听1个。

参与粉丝团福袋是付费行为，在加入粉丝团时，抖音官方风控系统有时会根据IP地址异常或其他原因，对付费行为检测比较严格。

在加入粉丝团时，小概率会将账号踢下线。请随时观察账号是否在线。账号被踢下线时，扭蛋机会发出声音报警。

最近工作比较忙，新功能开发比较缓慢，请大家见谅。

# 使用说明：

1.打开douyin_downloader软件目录，运行douyin_downloader.exe即可。

2.直播间抢福袋功能，支持扫描平台所有直播间，并过滤出正在发放福袋的直播间进行参与。

3.抢福袋可指定固定直播间，蹲点抢福袋功能，可以预先设置指定的直播间进行蹲点抢福袋。

4.支持查看中奖纪录，包括中奖时间，直播间地址，中奖物品的名称和中奖数量。新版本还增加了中奖提醒，中奖时会听到“叮”音效。

5.如果福袋中奖率过高，账号可能会被官方风控，一段时间内不会再中福袋（一般1-2天）。等一段时间再试试即可。

6.直播间抢福袋、HD高清录制、批量取关的功能，需要用户登录抖音账号。

7.直播间录制的MP4视频，有时候会出现进度条异常或黑屏的情况，可能是由于网络波动导致的视频文件不完整。使用MP4视频修复软件修复一下即可。

8.批量取关的博主数量如果过多，可能会被官方弹出验证码。请自行手动解决验证码问题。

9.支持http/https代理。在软件目录中配置proxy.txt即可。一行一个，格式为ip:port@user#password，无用户密码的代理可以留空，如8.8.8.8:8888@#
